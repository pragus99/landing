Pengenalan

PHP merupakan bahasa pemograman yang digunakan untuk membuat aplikasi berbasis website. Oleh karena itu, PHP dapat dijalankan mengggunakan browser.

PHP setidaknya digunakan karena sifatnya dinamis dan interaktif. Dinamis berarti bisa berganti konten sesuai kondisi tertentu. Sedangkan interaktif berarti website tersebut bisa memberikan feedback bagi user.

Syntax

Sebaiknya mengenal syntax-syntax dasar yang dikenal di lingkungan PHP. Menulis komentar dalam PHP digunakan sebagi catatan pribadi atau pesan untuk programmer lain. Titik koma untuk mengakhiri setiap menulis baris perintah dalam PHP kecuali baris yang berfungsi membuat blok. Simbol $ digunakan untuk mendeklarasikan variabel dan harus ditempat di depan setiap tipe variabel.

Berikut ini merupakan aturan-aturan syntax dasar dalam menulis dasar PHP:

  • Menulis komentar satu baris dan lebih dari satu baris
  • Simbol $
  • Titik koma

    • Contoh:

        // ini untuk satu baris

    */
      ini untuk lebih dari satu baris
      ini untuk lebih dari satu baris
    */

  $angka = 3;   //Simbol $ sebagai deklarasi setiap awal Variabel
  $string = "huruf";    //Titik koma selalu diakhir perintah.
  $array = array("Ford", "Hyundai", "Toyota");

Variabel

Variabel merupakan memori sementara untuk menyimpan data numerik maupun string. Harus diawali deklarasi simbol $.

Sebagai awal ini, ada tiga variabel yaitu:

  • Numerik
    • $angka = 10;
  • String
    • $huruf = "kucing";
  • Array
    • $array = array("kuntilanak", "gunduruwo", 3)

Variabel bersifat dinamis. Tidak harus mendeklarasikan variabel sebelum mengggunakannya dan tipe variabel dikonversi otomatis sesuai tipe data konteksnya.

Array

Array adalah variabel yang dapat menyimpan beberapa nilai sekaligus.

Jika dirangkum, Sebuah variabel array dapat:

  • Menyimpan banyak nilai dalam bentuk numerik atau string sekaligus.
  • Menyimpan nilai dari variabel array lainnya.

Penulisan beberapa variabel:

$organ = "hati"; $organ2 = "jantung"; #organ3 = "lambung";

Bisa disederhanakan dengan variabel array menjadi:

$organ = array("hati", "jantung", "lambung");

Function

Function merupakan kumpulan kode program yang dikemas. Tujuan utamanya membantu programmer melakukan tugas yang sama berulang-ulang tanpa harus menulis kode yang sama lagi.

Function dapat menghasilkan sebuah nilai. Biasanay disebut Return Value atau nilai balikan.


Hasil yang keluar adalah 20
